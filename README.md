# scorekeeper
A very boring scorekeeping website for two players which updates the score of each player as the match progresses. The reset button resets the match, setting the score to 0-0 while the drag down allows you to select the number of points played for. Languages used: HTML and Javascript. CSS Framework used: Bulma
